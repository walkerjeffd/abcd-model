Instructions

  1. Enter watershed name
  2. Enter watershed latitude in degrees North (needed to compute potential evapotranspiration)
  3. Drag and drop CSV input data file from your desktop
  4. Click Save button

Input Data Format

Input data must be in a comma-separated value (CSV) file format with the following columns:

  • Date: Date in format "yyyy-mm-dd" (e.g. 2014-03-10)
  • Precip_in: Precipitation (inches/day)
  • Tmin_degC: Minimum air temperature (degrees C)
  • Tmax_degC: Maximum air temperature (degrees C)
  • Flow_in: Streamfow (inches/day)

Beware!

If using Excel, make sure the date format is set correctly. The default "mm/dd/yyyy" format cannot be used. To specify the correct date format, highlight the date column and set the cell format to Custom with type "yyyy-mm-dd", then save the file as a CSV.
